# 🎬 כתוביות עברית להרצאות מוקלטות

אפליקציה שמתמללת הרצאות (Moodle / כל וידאו) לעברית ומציגה כתוביות **מסונכרנות בול לאודיו**. רץ מקומית על המחשב שלך — בלי ענן, בלי חשבונות, בלי עלות.

> מבוסס Whisper עם מודל עברי ייעודי (ivrit-ai). אומת: מתמלל עברית במדויק, כולל פיסוק.

---

## ⭐ הדרך הקלה — האפליקציה (בלי שורת פקודה)

הרצאות הן סרטונים **מוקלטים**, אז מתמללים אותן **פעם אחת** ומקבלים כתוביות מסונכרנות מצוין — גם בלי כרטיס מסך חזק.

1. **הורד את ההרצאה** מ-Moodle למחשב (קובץ וידאו).
2. **הפעל את האפליקציה** — לחיצה כפולה על קיצור הדרך **"כתוביות עברית"** בשולחן העבודה (או על `run.bat` בתיקייה). בפעם הראשונה היא מתקינה לבד את מה שצריך.
3. בחלון: **בחר קובץ הרצאה** → **צור כתוביות** → המתן → **▶ צפה עם כתוביות**.

זהו. הכל בכפתורים. 🎯

> דרישה חד-פעמית: **Python 3.12** מותקן (אם חסר, האפליקציה תפתח את עמוד ההורדה). הורדת המודל העברי (~1.5GB) קורית פעם אחת אוטומטית.

## למתקדמים

<details>
<summary>הרצה משורת פקודה / שילוב בנגן אחר (VLC)</summary>

```bash
cd backend
py -3.12 -m venv .venv && .venv\Scripts\activate    # macOS/Linux: python3.12 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python transcribe_to_srt.py "הרצאה.mp4"             # יוצר הרצאה.srt
```
לצפייה: פותחים את הווידאו ב-VLC (אם ה-`.srt` ליד הווידאו באותו שם — נטען אוטומטית), או ב-[`tools/player.html`](tools/player.html).
</details>

<details>
<summary>כתוביות חיות בדפדפן (תוסף Chrome — ניסיוני, מומלץ עם GPU)</summary>

על CPU התמלול החי לא מדביק את הקצב; מומלץ רק עם GPU.
1. `cd backend && python server.py` (שרת על `ws://localhost:9090`).
2. `chrome://extensions` → Developer mode → Load unpacked → תיקיית `extension/`.
3. פתח דף עם וידאו → אייקון התוסף → "▶ הפעל כתוביות".

פרטים ופתרון בעיות: [backend/README.md](backend/README.md). ארכיטקטורה: [BUILD_SPEC.md](BUILD_SPEC.md) · [CONTRACTS.md](CONTRACTS.md).
</details>

## 📁 מבנה

```
app.py                   ⭐ האפליקציה (חלון גרפי)
run.bat                  משגר האפליקציה (התקנה אוטומטית בפעם הראשונה)
backend/
  transcribe_to_srt.py   מנוע: קובץ הרצאה → SRT עברי מסונכרן
  server.py              שרת תמלול חי (לתוסף) + mock_server.py
tools/player.html        נגן מקומי לצפייה עם כתוביות
extension/               תוסף Chrome לכתוביות חיות
```
