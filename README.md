# 🎬 כתוביות עברית להרצאות מוקלטות

אפליקציה שמתמללת הרצאות (Moodle / כל וידאו) לעברית ומציגה כתוביות **מסונכרנות בול לאודיו**. רץ מקומית על המחשב שלך — בלי ענן, בלי חשבונות, בלי עלות.

> מבוסס Whisper עם מודל עברי ייעודי (ivrit-ai). אומת: מתמלל עברית במדויק, כולל פיסוק.

---

## ⭐ איך משתמשים (בלי שורת פקודה)

1. **הורד את ההרצאה** מ-Moodle למחשב (קובץ וידאו).
2. **הפעל את האפליקציה** — לחיצה כפולה על קיצור הדרך **"כתוביות עברית"** בשולחן העבודה (או על `run.bat`). בפעם הראשונה היא מתקינה לבד את מה שצריך.
3. בחלון: **גררו או בחרו** קובץ הרצאה → **צור כתוביות** (מציג שלב + זמן משוער) → **▶ צפה עם כתוביות**.

זהו. הכל בכפתורים. 🎯

> דרישה חד-פעמית: **Python 3.12** מותקן (אם חסר, האפליקציה תפתח את עמוד ההורדה). הורדת המודל העברי (~1.5GB) קורית פעם אחת אוטומטית. למחשב איטי יש מתג **"מהיר"**.

---

## למתקדמים — שורת פקודה

<details>
<summary>תמלול ישיר ל-SRT (לשילוב ב-VLC או בכל נגן)</summary>

```bash
py -3.12 -m venv .venv && .venv\Scripts\activate    # macOS/Linux: python3.12 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python tools/transcribe_to_srt.py "הרצאה.mp4"        # יוצר הרצאה.srt
```
לצפייה: פותחים את הווידאו ב-VLC (אם ה-`.srt` ליד הווידאו באותו שם — נטען אוטומטית), או ב-[`tools/player.html`](tools/player.html).
</details>

## 📁 מבנה

```
app.py                      משגר חלון pywebview + bridge ל-Python
engine.py                   מנוע התמלול (faster-whisper / ivrit-ai)
ui/                         ממשק המשתמש (HTML/CSS/JS + פונטים מקומיים)
run.bat                     משגר (התקנה אוטומטית בפעם הראשונה)
requirements.txt            תלויות
tools/
  transcribe_to_srt.py      תמלול משורת פקודה: קובץ → SRT
  player.html               נגן מקומי לצפייה עם כתוביות
```
