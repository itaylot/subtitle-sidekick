# 🎬 כתוביות עברית להרצאות מוקלטות

מתמלל הרצאות (Moodle / כל וידאו) לעברית ומציג כתוביות **מסונכרנות בול לאודיו**. רץ מקומית על המחשב שלך — בלי ענן, בלי חשבונות, בלי עלות.

> מבוסס Whisper עם מודל עברי ייעודי (ivrit-ai). אומת: מתמלל עברית במדויק, כולל פיסוק.

## ⭐ דרך A — תמלול הרצאה לקובץ כתוביות (מומלץ)

הרצאות הן סרטונים **מוקלטים**, אז עדיף לתמלל אותן **פעם אחת מראש** ולקבל כתוביות מסונכרנות בדיוק לאודיו. ככה אין לחץ של זמן-אמת, והאיכות והסנכרון מצוינים גם על מחשב רגיל.

### 1. הורד את ההרצאה
הורד את קובץ הווידאו של ההרצאה מ-Moodle למחשב (כפתור הורדה בנגן / קליק ימני / וכו').

### 2. התקן את מנוע התמלול (פעם אחת)
דורש **Python 3.11 או 3.12** (לא 3.14).
```bash
cd backend
py -3.12 -m venv .venv          # Windows;  macOS/Linux: python3.12 -m venv .venv
.venv\Scripts\activate          # macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
```

### 3. תמלל את ההרצאה
```bash
python transcribe_to_srt.py "הרצאה.mp4"
# יוצר "הרצאה.srt" — כתוביות עברית עם חותמות זמן מדויקות
```
> בהרצה ראשונה יורד המודל העברי (~1.5GB). על CPU התמלול לוקח זמן — רץ פעם אחת ושומר את הקובץ. למחשב איטי: `$env:MHS_MODEL="small"` (מהיר יותר, פחות מדויק).

### 4. צפה עם הכתוביות
- **VLC (או כל נגן):** פתח את קובץ הווידאו. אם ה-SRT נמצא ליד הווידאו באותו שם — הוא נטען אוטומטית, מסונכרן.
- **או בלי להתקין כלום:** פתח את [`tools/player.html`](tools/player.html) בדפדפן, בחר את הווידאו ואת ה-SRT, וצפה. הכל מקומי.

## דרך B — כתוביות חיות בדפדפן (תוסף Chrome, ניסיוני)

תוסף שמציג כתוביות חיות תוך כדי צפייה בדף. **מומלץ רק עם GPU** — על CPU התמלול לא מדביק את הקצב והכתוביות מאחרות מאוד.

1. הרץ שרת תמלול: `cd backend && python server.py` (מאזין על `ws://localhost:9090`).
2. טען את התוסף: `chrome://extensions` → Developer mode → Load unpacked → תיקיית `extension/`.
3. פתח דף עם וידאו → אייקון התוסף → "▶ הפעל כתוביות".

פרטים, GPU, ופתרון בעיות: [backend/README.md](backend/README.md). הארכיטקטורה: [BUILD_SPEC.md](BUILD_SPEC.md) · [CONTRACTS.md](CONTRACTS.md).
